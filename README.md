# Cilj igre
- Cilj igre je, da zbereš čim več točk s pobiranjem jabolk. Igralec upravlja roza kačo na svetlo modri podlagi.
- Ko kača poje navadno rdeče jabolko, se njeno telo podaljša, hitrost premikanja se postopoma povečuje, igralec pa dobi 1 točko.
- Občasno se na igralnem polju pojavi tudi posebno zlato jabolko, ki prinaša kar 5 dodatnih točk, vendar po določenem času izgine, če ga ne pobereš dovolj hitro.
- Igra se zaključi (Game Over), če se kača zaleti v rob zaslona, v svoje lastno telo ali pa v temno sive ovire (zidove), ki so naključno razporejene po igrišču. Po koncu igre se na zaslonu izpiše tvoj končni rezultat.
