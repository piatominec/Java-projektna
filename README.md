# POTEK IGRE
- Mi smo kača, ki zbira točke tako, da poje sadje, ki je enake barve kot kača in se s tem veča (kača postaja daljša)
- Barva kače se spreminja (igrica nas 3 sekundu prej opozori o spremembi barve)
- Vmes se pojavi zlato jabolko, ki ga lahko kača poje ne glede na barvo in pridobi dodatnih 5 točk
- Vmes se pojavijo nasprotniki (manjše črne kače) katerih se ne smemo dodakniti
## Izgubimo
- Če pojemo sadje drugačne barve kot je kača
- Če se tako zapletemo, da se zaletimo sami vase
- Če se nas dotakne nasprotnik
